02
Key Features
🧠
Double Dueling DQN
Separates Value V(s) and Advantage A(s,a) streams. Double DQN update decorrelates action selection from evaluation. LayerNorm + gradient clipping for stable training.
📈
Curriculum Learning
Auto-promotes easy → medium → hard when 75% success rate sustained over 30 episodes. Trains smarter, not longer. 1500 episodes to convergence.
⚡
REST API + Swagger
Full FastAPI server with auto-generated OpenAPI docs. Reset episodes, apply actions, read observations — all over HTTP. Compatible with any agent.
🎯
Reward Shaping
Per-step health bonuses reward maintaining good metrics. Action rewards: restart +0.35, rollback +0.40, scale_up +0.25. Escalate penalised at -0.80.
🔀
Stochastic Environment
Gaussian noise on CPU, latency, error_rate each step. Cascading failures if API is down. Realistic simulation of production system behaviour.
🤖
LLM + Heuristic Agents
Three inference modes: trained DQN, OpenAI-compatible LLM agent, and priority-rule heuristic baseline. Graceful fallback chain on errors.
